# blog-website

EJS와 mongoose, node JS를 이용해 간단한 블로그 웹 사이트를 제작했습니다.

# 기능
1. 기본 적인 글 쓰기가 가능합니다.
2. 글의 길이가 100자가 넘으면 뒤에 있는 글의 내용을 자세히 볼 수 있도록 하는 기능이 있습니다.
3. 글의 길이가 길 경우 자세히 보기 기능이 있습니다.
